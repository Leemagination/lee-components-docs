vue文件
<script setup>
import { LeePagination } from "lee-vue3-components";
import { ref } from "vue";

const currentPage = ref(1)
const currentSize = ref(10)
</script>

<template>
  <div>
    <div>当前页码: {{ currentPage }}</div>
    <div>当前分页数量: {{ currentSize }}</div>
    <LeePagination
      :pageCount="9"
      v-model:page="currentPage"
      v-model:pageSize="currentSize"
      :pageSizeOptions="[10,20,50,100,1000]"
      jump-text="立马跳转"
      showQuickJumper
      showSizePicker>
    </LeePagination>
    <br>
    <div>禁用状态</div>
    <LeePagination
      disabled
      :pageCount="9"
      v-model:page="currentPage"
      v-model:pageSize="currentSize"
      :pageSizeOptions="[10,20,50,100,1000]"
      showQuickJumper
      showSizePicker>
    </LeePagination>
  </div>
</template>

API

Pagination props

名称类型默认值说明
pagenumber1当前页码的值
pageSizenumber10分页数量的值
pageCountnumber1总页数
disabledbooleanfalse是否禁用
showSizePickerbooleanfalse是否展示分页数量选择器
pageSizeOptionsnumber[][10]分页数量可选数组
displayCountnumber9页码展示的个数
showQuickJumperbooleanfalse是否展示快速跳转
jumpTextstring跳转是否展示快速跳转
