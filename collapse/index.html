折叠面板 Collapse

折叠大段的文字，将页面空间让给其他更核心的内容

基础

演示
点开看看呗
再点点呗
最后点一下嘛
可以不点了
vue文件
<script setup>
import { LeeCollapse, LeeCollapseItem } from "lee-vue3-components";
</script>

<template>
  <div>
    <LeeCollapse>
      <LeeCollapseItem title="点开看看呗">一段文字</LeeCollapseItem>
      <LeeCollapseItem title="再点点呗">第二段文字</LeeCollapseItem>
      <LeeCollapseItem title="最后点一下嘛">第三段段文字</LeeCollapseItem>
      <LeeCollapseItem title="可以不点了">第四段段文字</LeeCollapseItem>
    </LeeCollapse>
  </div>
</template>

手风琴模式

演示
手风琴模式
点点呗
点一下嘛
默认展开
第四段文字
vue文件
<script setup>
import { LeeCollapse, LeeCollapseItem } from "lee-vue3-components";
</script>

<template>
  <div>
    <LeeCollapse accordion>
      <LeeCollapseItem title="手风琴模式" name="1">一段文字</LeeCollapseItem>
      <LeeCollapseItem title="点点呗" name="2">第二段文字</LeeCollapseItem>
      <LeeCollapseItem title="点一下嘛" name="3">第三段文字</LeeCollapseItem>
      <LeeCollapseItem title="默认展开" default-expanded name="4">第四段文字</LeeCollapseItem>
    </LeeCollapse>
  </div>
</template>

</style>

API

CollapseItem props

名称类型默认值说明
titlestring''折叠项标题
namestringSymbol折叠项的唯一name
defaultExpandedbooleanfalse是否默认展开

Collapse props

名称类型默认值说明
accordionbooleanfalse是否手风琴模式
