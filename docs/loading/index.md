# API
#### Loading props
| 名称         | 类型       | 默认值 | 说明         |
|------------|----------|----|------------|
| radius     | `number` | `6` | 圆圈半径       |
| speed     | `number` | `6` | 转一圈需要的时间(s) |
| width     | `number` | `6` | 圆圈宽度       |
| color     | `string` | `currentColor` | 颜色         |
