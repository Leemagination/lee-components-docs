# API
#### DatePicker props
| 名称          | 类型        | 默认值         | 说明     |
|-------------|-----------|-------------|--------|
| value       | `number`  | `undefined` | 选择器的值  |
| placeholder | `string`  | `'请选择日期'`   | 默认展示文案 |
| showTime       | `boolean` | `false`     | 是否可选时间 |
| disabled       | `boolean` | `false`     | 是否禁用状态 |
| clearable       | `boolean` | `false`     | 是否可清除  |
