# API
#### TimePicker props
| 名称         | 类型         | 默认值     | 说明      |
|------------|------------|---------|---------|
| value     | `number[]` | `null`  | 时间选择器的值 |
| placeholder     | `string`   | `请选择时间` | 默认文案    |
| disabled     | `boolean`  | `false` | 是否禁用    |
| clearable     | `boolean`  | `false` | 是否可清空   |
