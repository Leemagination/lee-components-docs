加载 Loading

显示加载中的图标,让用户更愿意等待

演示
基本
增加半径
调整转圈速度
圆圈宽度
修改颜色
vue文件
<script setup>
import { LeeLoading } from "lee-vue3-components";
</script>

<template>
  <div>
    <div>基本</div>
    <LeeLoading></LeeLoading>
    <div>增加半径</div>
    <LeeLoading :radius="20"></LeeLoading>
    <div>调整转圈速度</div>
    <LeeLoading :speed="2" :radius="15"></LeeLoading>
    <div>圆圈宽度</div>
    <LeeLoading :width="6" :radius="15"></LeeLoading>
    <div>修改颜色</div>
    <LeeLoading color="green" :radius="10"></LeeLoading>
  </div>
</template>

API

Loading props

名称类型默认值说明
radiusnumber6圆圈半径
speednumber6转一圈需要的时间(s)
widthnumber6圆圈宽度
colorstringcurrentColor颜色
